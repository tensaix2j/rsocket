rsocket
=======
Some socket server scripts for immediate use.

Ruby
-----
  Usage:
  
    ruby rsocket.rb <port>
  
  Example:
  
    ruby rsocket.rb 10000




Tcl
----
  Usage:

    tclsh tclsocket.tcl <port>

  Example:
  
    tclsh tclsocket.tcl 10000

MSVC++
-------
  
  Compile winsocket.cpp under MSVC++. 
  


WebSocket
---------
  Usage:
    ruby ws_rsocket.rb -port <port>
