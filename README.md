rsocket
=======
Some socket server scripts for quick use.

Ruby
-----
  Usage:
  
    ruby r_tcp.rb <port>
    ruby r_udp.rb <port>
  
  Example:
  
    ruby r_tcp.rb 10000
    ruby r_udp.rb 10000




Tcl
----
  Usage:

    tclsh tcl_tcp.tcl <port>
    tclsh tcl_udp.tcl <port>

  Example:
  
    tclsh tcl_tcp.tcl 10000
    tclsh tcl_udp.tcl 10000
    
    
MSVC++
-------
  
  Compile win_tcp.cpp under MSVC++. 
  


WebSocket
---------
  Usage:

    ruby ws_rsocket.rb -port <port>
